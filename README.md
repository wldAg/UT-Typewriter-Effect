# UT-Typewriter-Effect / UT打字机字幕

一个基于 UT 打字机实现的字幕程序

## 功能特点

- 实现屏幕上UT打字机逐字显示效果
- 支持字幕速度调节，修改颜色等

## 使用方法
你需要exe程序和Raylib.dll(自带)，并且自行创建Font文件夹(自带)和Run.txt文件(可以改名字，默认Run.txt)
程序是从外部文件读取命令(类似于脚本)的
按F3+ESC可以中途退出程序(全局监听,注意快捷键冲突)

#### 命令行启动
如果你使用的启动文件为Run.txt，则直接运行exe启动
如果使用的其它文件名(text.txt,Main,Main.run,...)则需要使用 程序 [文件名] 来启动程序，不然会报错或打开默认的Run.txt文件

#### 直接启动
准备一个Run.txt文件，然后启动程序

## 开发方法

1. 下载本项目代码
2. 还需要Visual Studio(本人用的2026),.Net 10
3. 用Visual Studio打开 基于UT文本引擎的字幕 by 无聊的Ag.slnx 文件，或直接双击 基于UT文本引擎的字幕 by 无聊的Ag.slnx 文件
4. 你需要通过nuget包管理来安装Raylib-cs(或者已经安装了)
5. 编译可以使用Visual Studio中的debug或Release编译，或者使用我提供的AOT编译中的AOT.bat编译(注意：AOT编译时间(20\~30s)>>>Visual Studio的简单编译(0.8\~1.5s))
